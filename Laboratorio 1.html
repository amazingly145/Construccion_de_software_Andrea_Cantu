Preguntas a responder:

1. ¿Cuál es la diferencia entre Internet y la World Wide Web?
Respuesta: El internet es una inmensa red de computadoras conectadas entre si, mientras que la World Wide Web (WWW) es una enorme colección de páginas que se sienta sobre esa red de computadoras
2. ¿Cuáles son las partes de un URL?
Respuesta: Una URL (Localizador Universal de Recursos) es un tipo de URI (Identificador Universal de Recuros) específico, un URL se ubica normalmente en un recurso ya existente de internet, el mismo se utiliza cuando un cliente web hace una petición al servidor para obtener un recurso.
3. ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
Respuesta: Los métodos HTTP, son el formato de comunicación entre el cliente y el servidor web.
- GET: realiza una petición a un recurso en específico.
- POST: puede enviar datos por medio del cuerpo
