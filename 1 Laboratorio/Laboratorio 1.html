2. ¿Cuáles son las partes de un URL?
Respuesta: Una URL (Localizador Universal de Recursos) es un tipo de URI (Identificador Universal de Recuros) específico, un URL se ubica normalmente en un recurso ya existente de internet, el mismo se utiliza cuando un cliente web hace una petición al servidor para obtener un recurso. El URL en el inicio tiene el prrotocolo, seguido por el dominio, el path y el parametro.
3. ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
Respuesta: Los métodos HTTP, son el formato de comunicación entre el cliente y el servidor web.
Los metodos mas comunes son GET y POST: - GET: realiza una petición a un recurso en específico. Los datos no son modificados ni los estados de los recursos
- POST: creamos un nuevo recurso, necesita de un cuerpo en donde defines la identidad de la data creada, puede enviar datos por medio del cuerpo para crear el recurso.
- PUT: usas este metodo para actualizar un recurso existente, utilizando el contenido del mismo, es una forma de "editar" algo. Si no hay un recurso que sea igual al que se pide, se crea uno nuevo.
- HEAD: usas este metodo igual que GET, pero solo regresa HEAD, no debe de tener el cuerpo, usas el metodo par verificar si el recurso esta presente a la hora de usar GET.
- PATCH: Usas el metodo para aplicar parcialmente las modificaciones del recurso, con patch solo necesitas pasar la data que quieres modificar
- DELETE: Usas el metodo para eliminar el recurso especificado.

6. Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?
Respuesta: Cuando un servidor web regresa el codigo 200, significa que la solicitud se hizo exitosamente. Lo cual signiifica que para los metodos:
- GET: los recursos han sido obtenidos y transmitidos en el mensaje del cuerpo.
- HEAD: Los headers se han incluido en la respuesta sin un mensaje del cuerpo.
- PUT o POST: Los recursos se han transmitid en el mensaje del cuerpo.
En resumen: No significa que hay un error, por lo contrario que se hizo exitosamente lo pedido.
